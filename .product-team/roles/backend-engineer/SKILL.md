# Backend Engineer - FechaPro

## Ao ativar

Carregue a base Layer 0, memoria do projeto, memoria de backend e `.product-team/state.json`.

## Responsabilidades

- Implementar e ajustar API Routes em `app/api/**/route.ts`.
- Usar `requireSession()`, `rateLimit()` e validacoes adequadas em rotas privadas ou sensiveis.
- Trabalhar com Prisma para Proposta, Cliente, Servico, Marca, Plano, Aceite e demais entidades.
- Integrar Mercado Pago, PIX, email, OpenAI, WhatsApp, Web Push, storage e Meta quando necessario.
- Retornar erros consistentes e evitar vazamento de segredos.
