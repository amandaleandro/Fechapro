# FechaPro

O FechaPro e uma plataforma comercial para prestadores de servico criarem propostas profissionais, acompanharem o interesse do cliente e venderem com mais confianca.

Em vez de mandar um orcamento simples pelo WhatsApp, o profissional entrega uma experiencia completa: link publico bonito, PDF, marca, portfolio, depoimentos, aceite online, pagamento e materiais de divulgacao.

## Posicionamento Comercial

**Promessa:** transformar atendimento, orcamento e fechamento em uma estrutura profissional que aumenta valor percebido antes da conversa sobre preco.

**Para quem:** prestadores de servico, pequenos negocios, profissionais autonomos, consultores, tecnicos, agencias, estetica, eventos, saude, educacao, manutencao, reformas e qualquer negocio que vende por proposta.

**Valor para o cliente final:**

- entende melhor o que esta comprando;
- ve escopo, prazo, valor, condicoes e proximos passos com clareza;
- confia mais porque recebe marca, portfolio, depoimentos e documento profissional;
- consegue aceitar, recusar, pagar ou chamar no WhatsApp pelo proprio link;
- recebe PDF e comprovantes comerciais mais organizados.

**Valor para quem vende:**

- para de depender de mensagens soltas no WhatsApp;
- cria propostas mais rapido usando clientes, servicos, templates e pacotes;
- sabe quem abriu, quem clicou no WhatsApp, quem aceitou, quem recusou e quem pagou;
- faz follow-up com contexto;
- vende com mais autoridade visual;
- divulga servicos com artes prontas para post, story e status;
- centraliza proposta, portfolio, depoimentos, pagamentos, suporte e plano em um painel.

## Oferta Comercial

O FechaPro deve ser apresentado como uma estrutura comercial pronta, nao apenas como um sistema de orcamento.

Em vez de vender "sistema de orcamento", a oferta principal e:

**"Uma estrutura comercial pronta para sua empresa vender com mais profissionalismo no WhatsApp, Instagram e propostas."**

O cliente compra:

- sistema FechaPro;
- proposta profissional com link, PDF, aceite e pagamento;
- presenca digital melhorada;
- mini site profissional no pacote completo;
- materiais comerciais;
- mensagens prontas;
- implantacao assistida;
- treinamento rapido.

Mensagem central:

**"A gente organiza sua apresentacao comercial para voce parar de mandar orcamento simples e comecar a vender com mais autoridade."**

## Recursos Que Agregam Valor

### Propostas profissionais

- criacao e edicao de propostas com cliente, servico, valor, prazo, validade, escopo, observacoes e forma de pagamento;
- link publico para enviar no WhatsApp, Instagram, e-mail ou qualquer canal;
- PDF profissional da proposta;
- aceite online com nome e e-mail;
- recusa com motivo;
- duplicacao e reenvio de propostas;
- status de rascunho, enviada, visualizada, aguardando resposta, aceita, recusada e expirada;
- escolha de tipo de documento: orcamento, proposta comercial, proposta tecnica, plano de cuidado ou proposta de evento;
- escolha de segmento visual para adaptar proposta e PDF ao nicho.

### Acompanhamento comercial

- painel com indicadores de propostas, valor aberto, aceites, visualizacoes e cliques no WhatsApp;
- contagem de visualizacoes da proposta publica;
- contagem de cliques no WhatsApp;
- detalhes de aceite, recusa, pagamento e atualizacao;
- follow-up manual mais inteligente com base no comportamento do cliente;
- alertas por push para proposta visualizada, aceita, recusada ou paga;
- notificacoes por WhatsApp para o profissional quando configurado pelo admin.

### Pagamentos

- checkout de propostas pelo Mercado Pago;
- PIX direto para a chave do profissional com QR Code e copia e cola;
- confirmacao manual de PIX direto;
- checkout de planos pelo Mercado Pago;
- compra de creditos extras de artes pelo Mercado Pago;
- webhook para atualizar pagamentos integrados.

### Marca e autoridade

- cadastro de logo, cores, contatos, WhatsApp, e-mail, Instagram, site e chave PIX;
- textos comerciais de introducao, fechamento, termos e perguntas frequentes;
- controle de exibicao de portfolio, depoimentos, servicos e FAQ;
- upload de logo com remocao automatica de fundo claro;
- portfolio com imagens e categorias;
- depoimentos de clientes;
- biblioteca de servicos com preco, prazo e itens inclusos.

### Templates, pacotes e nichos

- templates prontos por nicho;
- criacao de modelos recorrentes pelo usuario;
- importacao de templates;
- pacotes com varios servicos na mesma proposta;
- segmentos para organizar clientes e adaptar a comunicacao.

### Artes de divulgacao

- solicitacao de artes para Instagram post, Instagram story e status do WhatsApp;
- briefing guiado por objetivo: servico, promocao ou produto;
- envio de referencias visuais e fotos;
- opcao de usar imagem enviada como fundo;
- legenda e mensagem de WhatsApp junto com a arte;
- fluxo de aprovacao e download da arte final;
- creditos mensais por plano e pacotes extras.

### Atendimento, suporte e administracao

- cadastro, login, recuperacao e redefinicao de senha;
- onboarding guiado para configurar marca, servico e primeira proposta;
- importacao de clientes, servicos e depoimentos;
- aba de suporte para mensagens entre usuario e equipe;
- painel admin com usuarios, metricas, suporte, artes solicitadas e controle de assinaturas;
- criacao de usuarios pelo admin sem checkout;
- propostas demo para demonstracao comercial.

## Stack

- **Aplicacao:** Next.js, React e TypeScript.
- **UI:** Tailwind CSS e Lucide Icons.
- **Banco:** Postgres.
- **ORM:** Prisma.
- **PDF:** PDFKit.
- **Imagens:** Sharp.
- **Pagamentos:** Mercado Pago.
- **E-mail:** SMTP ou Resend.
- **Push:** Web Push com VAPID.
- **Observabilidade:** Sentry e healthcheck.
- **Infra local:** Docker Compose.

## Desenvolvimento Local

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

Comandos uteis:

```bash
npm run lint
npm run test
npm run build
npm run db:generate
npm run db:push
npm run db:studio
npm run admin:create
```

## Rodando Com Docker

```bash
docker compose up --build -d
```

Servicos:

- App: `http://localhost:3000`
- Postgres: `localhost:5436`
- Database: `fechapro`
- User: `fechapro`
- Uploads locais: volume Docker `fechapro_uploads`

Para aplicar o schema:

```bash
docker compose run --rm migrate
```

Para criar ou atualizar admin:

```bash
docker compose exec app node scripts/create-admin.js
```

## Configuracao

Use `.env.example` como base. Configure principalmente:

- `DATABASE_URL`
- `AUTH_SECRET`
- `AUTH_URL`
- `NEXTAUTH_URL`
- `ADMIN_EMAILS`
- `MERCADO_PAGO_ACCESS_TOKEN`
- `MERCADO_PAGO_WEBHOOK_SECRET`
- `APP_URL`
- `NEXT_PUBLIC_APP_URL`
- `NEXT_PUBLIC_SITE_URL`
- `NEXT_PUBLIC_WHATSAPP_NUMBER`
- SMTP ou `RESEND_API_KEY`
- chaves VAPID para push
- S3/R2 para storage externo em producao
- Cloudflare Turnstile para formularios publicos
- Sentry e `HEALTHCHECK_TOKEN` se usar monitoramento

## Pagamentos

O checkout de planos, creditos de artes e propostas usa Mercado Pago. Configure o webhook em:

```text
https://seu-dominio.com/api/webhooks/mercadopago
```

Nas propostas, o profissional tambem pode escolher PIX direto. Nesse caso, o FechaPro mostra a chave configurada, QR Code e copia e cola; a confirmacao precisa ser combinada ou marcada manualmente.
